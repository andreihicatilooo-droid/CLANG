# Дизайн-система «Северный Кот» (NORTHCAT)

Извлечена из брендовых материалов тату-студии (готический постер, круглая эмблема, шрифтовые сеты). Стиль: **dark gothic / blackletter**, неоновый розовый на почти чёрном тёмно-зелёном фоне, гравюрная иллюстрация.

## 1. Палитра

Цвета замерены попиксельно из исходных изображений.

| Токен | HEX | Использование |
|-------|-----|---------------|
| `bg-void` | `#11140E` | Основной фон (почти чёрный, тёплый зелёный подтон) |
| `bg-deep` | `#13150F` | Фон секций / карточек |
| `bg-raised` | `#1B1A16` | Приподнятые поверхности, hover |
| `accent` | `#D1536D` | Главный акцент — «неоновый» розовый (текст, кнопки, ссылки) |
| `accent-bright` | `#F0607E` | Акцент при hover / выделение |
| `accent-deep` | `#A24457` | Затенённый акцент, нажатое состояние |
| `plum` | `#683650` | Вторичный — приглушённый сливово-фиолетовый (иллюстрации, иконки) |
| `plum-dark` | `#492235` | Тени иллюстраций, бордеры карточек |
| `line` | `#332229` | Тонкие линии, разделители, рамки |

Контрастная пара: `#D1536D` на `#11140E` ≈ 5.4:1 — достаточно для текста.

```css
:root {
  --bg-void: #11140E;
  --bg-deep: #13150F;
  --bg-raised: #1B1A16;
  --accent: #D1536D;
  --accent-bright: #F0607E;
  --accent-deep: #A24457;
  --plum: #683650;
  --plum-dark: #492235;
  --line: #332229;
}
```

## 2. Типографика

Двухуровневая система: готика для заголовков, гротеск для служебного текста.

| Роль | Стиль | Пример в макетах |
|------|-------|------------------|
| Display / H1–H2 | Blackletter (готический), UPPERCASE, плотный трекинг | «СЕВЕРНЫЙ», «NORTHCAT», «REAP» |
| H3 / подзаголовки | Тот же blackletter, меньший кегль | «Тату студия», «THE VOID» |
| Служебный текст, метки | Узкий гротеск, UPPERCASE, разреженный трекинг (letter-spacing ≈ 0.2em) | «NORTHCAT.TOP», «DESIGNED BY…», «GOTHIC CHRONICLE» |

Подходящие свободные шрифты:

- Заголовки: **Pirata One**, **UnifrakturCook**, **Germania One** (поддержка кириллицы — проверить; для кириллицы — «Fraktur RU»-подобные или кастомный сет из исходника `Dark Urbanic`)
- Гротеск: **Inter** / **IBM Plex Sans** с `text-transform: uppercase; letter-spacing: .18em`

Особенность исходного шрифта: два стилистических сета заглавных (Stylistic Set 1 / 2) — альтернативные формы букв для вариативности заголовков.

## 3. Графические мотивы

- **Гравюрная иллюстрация** (линогравюра/вудкат): кошка-сфинкс, песочные часы с черепом, рунический камень, закатное солнце. Заливки — `plum`/`plum-dark`, блики — `accent`.
- **Оккультная геометрия**: тонкие линии `line`/`plum`, пересекающиеся диагонали, круговые рамки (двойное кольцо у эмблемы), сигилы.
- **Текст по дуге** в круглых эмблемах (верх — латиница, низ — кириллица).
- **Зернистость/шум** на фоне — лёгкий grain поверх `bg-void`.
- **UI-маркеры в стиле «терминала»**: квадратные элементы с `×`, мелкие пиктограммы-руны, штрих-блоки.

## 4. Компоненты (применение к UI)

| Компонент | Рецепт |
|-----------|--------|
| Фон окна | `bg-void` + grain-текстура (opacity ~4%) |
| Карточка | `bg-deep`, рамка 1px `line`, радиус 4–6px (углы почти прямые) |
| Кнопка primary | Заливка `accent`, текст `bg-void`, hover → `accent-bright` |
| Кнопка secondary | Прозрачная, рамка 1px `accent`, текст `accent` |
| Заголовок экрана | Blackletter, UPPERCASE, цвет `accent` |
| Метки/капшены | Гротеск 11–12px, UPPERCASE, letter-spacing .18em, цвет `plum` → `accent` при активности |
| Разделители | 1px `line`, допустимы диагональные декоративные линии |
| Иконки | Однотонные, стиль гравюры/рун, цвет `plum`, активные — `accent` |

## 6. Тон бренда

Мистический, северный, «вечная тьма» — но с иронией. Слоганы в духе «REAP THE VOID», «VOID CRAFTED», «ETERNAL CREATIVITY». Дата/хроника как декоративный элемент (`GOTHIC CHRONICLE • 13/05/2026`). Домен `northcat.top` всегда мелким гротеском сверху по центру.
