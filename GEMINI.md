# Контекст проекта — Screen Translator

## Обзор проекта
Screen Translator — это десктопное приложение для Windows, написанное на Python. Оно позволяет пользователям выделять область на экране и мгновенно переводить текст внутри этой области. Для извлечения текста используется встроенный Windows OCR, а также поддерживаются несколько систем перевода, включая Google Translate и Google Gemini Vision API.

### Основные возможности
- **Захват области**: Оверлей для выбора области экрана, вызываемый горячей клавишей.
- **OCR (Оптическое распознавание символов)**: Использует `winrt` (Windows Runtime API) для быстрого встроенного распознавания текста без внешних зависимостей.
- **Движки перевода**:
  - Google Translate (текст-в-текст через `deep-translator`).
  - Gemini Vision API (перевод картинки в текст с использованием `gemini-2.5-flash`), доступен через API-ключ или OAuth.
- **Пользовательский интерфейс**: Использует `tkinter` для интерфейса настроек и оверлея выбора экрана, а также `pystray` для иконки в системном трее.
- **Конфигурация**: Хранится в `%APPDATA%\ScreenTranslator\config.json`.

## Структура директорий
- `screen_translator/screen_translator.py`: Главная точка входа в приложение. Инициализирует системный трей, слушатель горячих клавиш и пользовательский интерфейс.
- `screen_translator/app/`: Основной пакет приложения.
  - `config.py`: Управление постоянной конфигурацией в формате JSON.
  - `hotkey.py`: Регистрация и прослушивание глобальных горячих клавиш.
  - `capture.py`: Логика захвата экрана и выбора области с помощью оверлеев Tkinter.
  - `ocr.py`: Интеграция с `winrt.windows.media.ocr` для извлечения текста.
  - `translators.py`: API-интеграции для Google Translate и Gemini.
  - `settings_ui.py`: Диалоговое окно настроек на базе Tkinter.
  - `tray.py`: Реализация иконки системного трея.
- `screen_translator/requirements.txt`: Зависимости пакетов Python.
- `screen_translator/build.bat`: Скрипт сборки PyInstaller для компиляции приложения в автономный исполняемый файл Windows.

## Сборка и запуск

### Настройка для разработки
1. Убедитесь, что у вас установлен Python.
2. Установите необходимые зависимости:
   ```cmd
   pip install -r screen_translator/requirements.txt
   ```
3. Запустите приложение из исходного кода:
   ```cmd
   python screen_translator/screen_translator.py
   ```

### Сборка исполняемого файла
Запустите прилагаемый batch-скрипт, чтобы упаковать приложение в единый исполняемый файл с помощью PyInstaller:
```cmd
cd screen_translator
build.bat
```
Скомпилированный автономный исполняемый файл будет создан по пути `screen_translator/dist/ScreenTranslator.exe`.

## Соглашения по разработке
- **Платформа**: Приложение сильно зависит от специфичных для Windows API (`winrt` для OCR) и предназначено исключительно для Windows.
- **UI-инструментарий**: `tkinter` выбран в качестве стандарта для пользовательского интерфейса, чтобы минимизировать внешние зависимости.
- **Многопоточность**: асинхронные функции `winrt` обрабатываются в выделенных циклах событий внутри потоков, чтобы не блокировать основной цикл Tkinter (как реализовано в `app/ocr.py`).
